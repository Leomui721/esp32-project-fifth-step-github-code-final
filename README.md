# esp32-project-fifth-step-github-code-final
  本程式採用 MicroPython 語言開發，為 ESP32 智能長者照護系統之完整主控程式，整合即時時鐘（RTC）、RFID 藥品辨識、HX711 重量感測、ESP32‑CAM 影像拍攝、AI 跌倒偵測、LED 指示燈及蜂鳴器提醒等所有功能。
  程式採無窮迴圈運行，自動依序偵測各項感測器與觸發條件，無需手動操作，具備穩定、省電、反應快速等特性。
  當到達服藥時間、偵測到藥品 RFID、拿起藥盒，或是 AI 判斷發生跌倒時，系統會自動觸發對應的燈光與聲音提醒，並驅動相機進行拍照記錄，達到完整照護警示效果。
